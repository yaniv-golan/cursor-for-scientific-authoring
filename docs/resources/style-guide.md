# Style Guide

Define tone, terminology, and formatting conventions for the playbook here.

## Draft principles

- Write in clear, plain language with actionable steps.
- Highlight reproducibility considerations and ethical safeguards.
- Prefer Markdown tables and lists for checklists and workflows.
- Link to original sources and note licensing for borrowed material.
